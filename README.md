# Features
## Existing Features
Sports Quiz holds 20 questions at the moment. When a player starts the game, the code will select 20 questions at random. So every time a player begins a game, they will never be in the same order or questions. Before starting the quiz, the plyer should read the instructions page - icon COGS. After finishing the quiz, the player will get there score with a message saying Game Over. The website also allows players to watch sports clips through out the page from a number of different Icon links located in its footer and head.
